Microsoft team

Setting a Microsoft Team alerting is simple:

Pre-requisite#

  • 1 minute ðŸš€
  • Create a microsoft team webhook : Guide to create an incoming webhook on Microsoft Team
  • Valid RestQA config file.

Configuration#

environments:
- name: sandbox
outputs:
- type: 'microsoft-teams'
enabled: true
config:
url: !env-var MSTEAM_WEBHOOK_URL
onlyFailed: false
showErrors: true

Options#

PropertyDescriptionDefault
urlThe microsoft team webhook url
onlyFailedTrigger the hook only for test failuretrue
showErrorsShow the error message within discordfalse
reportUrlThe url to access to your detail test report if you have one ({uuid} is a placeholder)

About the config.url we recommend to use the !env-var keyword in order to use an environment variable and not expose a sensitve url into your configuration.
