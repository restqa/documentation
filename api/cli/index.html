Cli

Start a new restqa project#

Test Automation Workflows can not only be started by triggers, webhooks, or manually via the Editor. It is also possible to start them directly via the CLI.

Execute a saved workflow by its ID:

restqa init

Install a new plugin#

Retrieve the list of available plugin:

restqa install

If you want to install the slack plugin:

restqa install slack

example:

restqa install <PLUGIN>

Step definition#

To get the list of different steps available for your scenario

Given#

restqa steps given

When#

restqa steps when

Then#

restqa steps then

Run the test#

You can run the test by using the command

restqa run

A few options are available :

ParameterAliasDescriptionDefault
--config-cSpecify the configuration file to use.restqa.yml
--env-eSpecify the environement to pick from the configuration fileDefault environment in the config
